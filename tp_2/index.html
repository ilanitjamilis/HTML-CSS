Cosas que tal vez no sabías de HTML y CSS

Colores
Perú y Tomate son en realidad dos nombres de color CSS válidos. Además, el hex # badA55 color no es tan badass, es en realidad una especie de cal :) Además, cuando se utiliza el atributo HTML bgcolor obsoleto, se muestra un color rojo sangriento.

CSS puede ser #gracioso
Compruebe las siguientes líneas CSS para poner una sonrisa en su cara:
#universe: before { content: ""; }
.ninja { visibility: hidden; }
.kids-these-days { -webkit-perspective: none; }
.work:after{ content:"Beer!"; }
